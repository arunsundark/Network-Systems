


HTTP WEBSERVER

1. Install Instruction

* DO make build 
* Do ./server

2. TO remove executables

* Do make clean


3. Functions

This webserver supports HTTP GET requestion for the given webpage
Performs error handling and reports two errors.
* 404 FILE NOT FOUND
* 500 INTERNAL SERVER ERROR

4. To Exit
* Press ctrl + c

5. To run client 

* Do localhost:8081
